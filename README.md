# Carrot Market

## Tailwind

-   yarn add -D tailwindcss postcss autoprefixer
-   npx tailwindcss init -p

## PlanetScale

-   pscale auth login
-   pscale database create carrot-market --region ap-northeast
-   pscale connect carrot-market
-   npx prisma db push
-   npx prisma studio (관리자 페이지)
-   npx prisma generate
